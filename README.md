### Getting Started

#### Node Application

1. Open up a terminal and execute the following commands to install node dependencies.

```
cd node
npm install
```

2. Run the application

```
node node.js --config=./configs/dev.json
```

#### Web/REST API Application

1. Open up a terminal and execute the following commands to install node dependencies.

```
cd api
npm install
```

2. Run the application

```
npm run dev
```
